Project Currently Working On

Morel Compass

Morel Compass is a predictive mushroom habitat mapping system that combines terrain analysis, wildfire history, National Water Model variables, satellite vegetation indices, environmental feature engineering, and machine learning inference to estimate likely mushroom emergence conditions.

The embedded visualization below is a static export example from the current Washington State prediction environment. The full project operates as a dynamic Python-based system capable of much larger environmental datasets, temporal prediction layers, and multi-species habitat inference.
